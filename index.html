EdiWeave

An open-source, mutli-platform EDI framework for .NET

Parse and create X12, HIPAA, EDIFACT, EANCOM, VDA and PNRGOV documents with ease.

Download Source Code

Always Open Source

EdiWeave is a hard-fork of the now closed-source EdiFabric project.
EdiWeave will always be Open Source under the LGPL.

See the license.

Portable and Modern

EdiWeave supports modern standards.
